# 29 October Project

This project contains a React App built for 29 October Turkish Republic Day.

## Technologies

| Technology         | Version      |
| ------------------ | ------------ |
| React              | 17.0.2       |
| React Audio Player | 0.17.0       |
| Nuka Carousel      | 4.8.1        |
| React Bootstrap    | 2.0.0-beta.4 |
| Bootstrap          | 5.0.2        |
